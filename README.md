WEB GAME > THE SPACE LOG
=====================


**The space log is web strategy game.**

> - **NOTE:**

> - **Development Environment:** Linux(Ubuntu)
> - **Programming language:** html5, css, and javascript
> - **Library and Framework:** jquery, node.js, express, and socket.io 
> - **Database:** MongoDB 
> - **Version control:** Git
